Question 1: Qu'est-ce que HTML? ▼
HTML (HyperText Markup Language) est le langage standard pour créer des pages web. Il définit la structure de la page à l'aide de balises.
Question 2: Comment styliser une page web? ▼
Vous pouvez styliser une page web en utilisant du CSS (Cascading Style Sheets). Il permet de définir le style, la mise en page et l'apparence de votre site.
